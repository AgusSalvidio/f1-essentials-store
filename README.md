# F1 Essentials Store

<img src="assets/icon.png" alt="Logo" width="100" height="100">

F1 Essentials Store is an e-commerce project used for Coderhouse course. The application allows you to view a list of products and simulate online purchases. Products and purchase orders are stored in Firebase.

## Frameworks 🏗️
- React Native + Expo
- React Navigation
- Redux Toolkit 
- Firebase

## Features 📜
- View available products list
- Add products to shopping cart
- View shopping cart
- Place a purchase order
